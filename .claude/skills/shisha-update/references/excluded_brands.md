# 除外ブランドリスト

`data/shishaData.js` はシーシャ (水タバコ) のデータ。MOFの「パイプたばこ」区分には伝統的パイプタバコやシャグタバコ (手巻き) も含まれるので除外する。

実行時に使うパターンは [excluded_brands.json](excluded_brands.json) にある。このファイルは人間向けの解説。

## 除外カテゴリ

### 伝統的パイプタバコ
**西洋系**: ダビドフ, マックバレン, Samuel Gawith, Dr.Pipe, RAW, フォン・クラーレン, 4th ジェネレーション, Three Nuns, バルカン・フレイク, グロウスムーア, コモンウェルス, スクワドロン, フル・バージニア, ベスト・ブラウンフレイク, サムズ・フレイク, セントジェームス・フレイク, コーネルディール, ジーエル・ピース, ペペ (独), ゴールデンブレンド, チョイス, アクレマ, コルツ, ドーラ

**JT系 (日本たばこ)**: 飛鳥, 桃山, 桃山100II, プロムナード, シルクロード, ロックン・チェアII, カピート・メローライトII, セブンシーズ, ハイタバコ

**プエブロ系**: プエブロナチュラル (デンマーク、パイプタバコ兼シャグ)

**キャプテンブラック**: ブランド名は同じでシーシャ系もあり得るが、MOFに載る製品はパイプタバコ

### シャグタバコ (手巻きタバコ)
**ベルギー**: フランドリア / フラントリア, スタンレー・ダブル, レッドフィールド
**ラオス**: チェ・シャグ, ブラックスパイダーシャグ, ソル, TP-77, VK-05/06, キチャグ, JBR (30gパウチ/ドイツ製だがシャグ)
**その他**: オールドホルボーン (英), アムステルダマー (蘭), ゴールデンクローバー, アークローヤル

## 判定のヒント

新しいブランドが出てきたとき、以下を手がかりに判定する:

- **パッケージサイズ**: シーシャは 50g箱 / 250g・500g・1kg パウチ が主流。30g パウチは手巻き寄り
- **製造国**: シーシャ系 = UAE / ヨルダン / トルコ / ロシア / インドネシア / 日本 (Samurai Blond) / 台湾
- **パイプ/シャグ系** = ドイツ / デンマーク / ベルギー / イギリス / アメリカ / ラオス
- **価格帯**: シャグは 500〜1,500円、シーシャは 1,500〜5,000円が多い

迷ったらユーザーに確認する。無理に分類してノイズを増やすよりマシ。

## リスト更新時

新しい除外ブランドが見つかったら [excluded_brands.json](excluded_brands.json) を更新する。
- 一意の文字列で他の製品と被らないなら `substrings`
- 「チェ」「ペペ」のように汎用語っぽいものは `exact_or_prefix` (完全一致 or 先頭一致+区切り)
